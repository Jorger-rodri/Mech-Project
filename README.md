# Classical Mechanics Projects

This repository contains the computational projects developed for the **Classical Mechanics** course.

The goal of these projects is to use numerical methods to study different physical systems and compare their behavior with theoretical models. All simulations are implemented in Python using Euler's method to integrate the equations of motion.

## Project 1 - Euler Integration

The first project introduces Euler's method through simple one-dimensional motion problems.

Topics covered:

- Free-fall motion
- Numerical integration of velocity and position
- Comparison with analytical solutions
- Motion under linear drag forces

This project serves as an introduction to solving differential equations numerically and understanding the limitations and accuracy of Euler integration.

## Project 2 - Rocket Trajectory Simulation

The second project extends the numerical approach to a more realistic three-dimensional system: a rocket launched from Earth.

The simulation includes:

- Gravity
- Atmospheric drag
- Variable mass due to fuel consumption
- Coriolis force from Earth's rotation
- Altitude-dependent atmospheric density

Different parameters can be modified to explore how they affect the rocket's trajectory and overall motion.
